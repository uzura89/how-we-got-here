# Contributing to "How We Got Here"

We're thrilled that you're interested in contributing to our AI-generated human history timeline project! This document outlines how you can contribute and help improve our resource.

## Ways to Contribute

1. Reporting Inaccuracies
2. Suggesting Improvements
3. Improving Code
4. Enhancing Documentation

## Getting Started

1. Fork the repository on GitHub.
2. Clone your forked repository to your local machine.
3. Create a new branch for your contribution.

## Contribution Guidelines

### 1. Reporting Inaccuracies

If you spot any historical inaccuracies in our content:

- Open an issue on GitHub.
- Clearly describe the inaccuracy.
- Provide reliable sources that support the correction.

### 2. Suggesting Improvements

For general improvements to existing content:

- Open an issue describing your suggested improvement.
- Explain why you think this improvement is valuable.

### 3. Improving Code

For code contributions:

- Follow the existing code style and conventions.
- Write clear, commented code.
- Include tests for new features or bug fixes.

### 4. Enhancing Documentation

To improve our documentation:

- Open a pull request with your changes.
- Ensure your writing is clear and concise.

## Event Selection Policy

Please note that we do not accept changes for the selection of events themselves unless there is obvious misinformation that contradicts many reputable sources. Our event selection is based on AI-generated content and a specific methodology, which we aim to maintain for consistency.

## Pull Request Process

1. Ensure your code adheres to the project's style and standards.
2. Update the README.md or relevant documentation with details of changes, if applicable.
3. Increase the version numbers in any examples files and the README.md to the new version that this Pull Request would represent.
4. Your Pull Request will be reviewed by maintainers. Be open to feedback and be prepared to make changes if requested.

## Questions?

If you have any questions about contributing, feel free to open an issue for discussion.

Thank you for helping to improve "How We Got Here"! Your contributions play a crucial role in making this a valuable historical resource.
